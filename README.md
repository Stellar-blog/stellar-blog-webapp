# stellar-blog.com
> Fullstack webapp deployed on https://www.stellar-blog.com

It's a blog style social media webapp built with Next.js & GraphQL on AWS.

## Front-end development tools
- HTML 5, CSS 3, Javascript (ES6)
- Next.js (React.js)
- Styled Components

## Back-end development tools
- Node.js (v12)
- Express.js
- GraphQL

## Unit-test tools
- Jest
- Enzyme
- @testing-library

## AWS services
- AWS Amplify
- AWS Appsync
- AWS DynamoDB
- AWS S3
- AWS Congnito
- AWS Lambda
- AWS Route 53
- AWs CloudFront
